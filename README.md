# VidyarthiBus

VidyarthiBus is an Android application designed to help students manage and track bus transportation efficiently.

## Features

* Student bus tracking
* Route management
* Real-time updates
* User-friendly Android interface
* Secure authentication

## Tech Stack

* Kotlin / Java
* Android Studio
* Firebase (if used)
* Google Maps API (if used)

## Installation

1. Clone the repository

```bash
git clone https://github.com/Kavikiran/VidyarthiBus.git
```

2. Open the project in Android Studio

3. Sync Gradle files

4. Run the app on an emulator or Android device

## Project Structure

* `app/` → Main Android application
* `gradle/` → Build configuration
* `assets/` → App resources
